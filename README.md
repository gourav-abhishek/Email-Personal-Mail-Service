## Features

- **Authentication:** Login and signup with session persistence (localStorage).
- **Landing Page:** Marketing homepage with hero section and feature highlights.
- **Dashboard:** Gmail-style 3-pane layout with sidebar, email list, and email content.
- **Email Management:** Inbox, Starred, Sent, Drafts, Spam, Trash folders.
- **Email Actions:** Star/unstar, read/unread toggle, delete (move to Trash), restore.
- **Compose Modal:** Rich compose window to create/send mock emails.
- **Bulk Actions:** Select multiple emails for batch operations.
- **Search:** Live search filtering across sender, subject, and more.
- **Settings & Profile:** Customize app theme, language, and personal info.
- **Responsive Design:** Mobile-friendly with hamburger menu and stacked views.
- **Animations & UX:** Smooth transitions, toast notifications, polished UI.
- **Mock Data:** 60+ realistic emails with varied senders, content, timestamps.
